Necesito ayuda para desarrollar una aplicacion de escritorio en c++

El software es super sencillo necesito una aplicacion que tenga una base de datos sqlite para poder darle ingreso a productos y a sus ventas en una tienda. Actualmente se realiza el proceso de actualizacion de dichos productos de sus ventas manualmente en un excel. 

Actualmente se toma todos los vales de ventas ocurridas en todo el dia y se comienza a llenar un  excel de ventas, que contiene todos los productos de la tienda, para proceder con la actualizacion de las ventas ocurridas de los productos que ocurrieron en el dia en la tienda. 

Se quiere automatizar este proceso lo mas rapido posible con un software programado en c++ para abrir en pc en local para hacer este procedimiento. El software debe de contar con:
- reportes / impresion  de ventas ocurridas de los productos. 
- Gestion de Productos
- Gestion para la tasa de cambio monetaria 
- Gestion de Margen comercial
- Gestion para las ventas

en la gestion de productos debe de permitir ajustar manualmente precio de costo que se adquirió, ya que puede variar en cuestion de la diferencia de tasa de cambio monetaria en el tiempo, el producto en dolares USD, asi como tambien se puede manipular manualmente el precio de venta ya que pueden existir productos que se adquieren em Moneda Nacional (CUP) y se ajusta a una venta aproximada. 


La tabla de productos debe de contener nombre, descripcion, Precio de Costo (USD / MN), Margen, precio de venta (opcional).
monedas a utilizar solamente:
- Dolar Americano (USD)
- Moneda Nacional Cubana (CUP)

La tabla de venta debe de tener  un ID de venta, precio de venta, fecha de la venta

Debe de existir una tabla que sea para el precio del dolar que actualizara el precio de venta del producto

Necesito que el proyecto siga bases de SOLID y una arquitectura de organizacion basada en caracteristicas. No codigo repetido, separacion de responsabilidad clara.

La aplicación va a ser ligera y rápida con un estilo visual moderno en c++ muy funcional para Windows 10 y superior.
